# Project Overview: ivanovishado.github.io

## Purpose
A high-end, cinematic personal portfolio website with a "Galactic Odyssey" theme. It features a "scroll-scripted" journey where a starship navigates through career achievements ("galaxies").

## Tech Stack
- **Framework:** Next.js 16.1.3 (App Router with `src/app` structure)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 (using `@tailwindcss/postcss`)
- **UI Architecture:** HeroUI v3 (Beta) (`@heroui/react`, `@heroui/styles`) & Tailwind Variants
- **Animation:** Motion (formerly Framer Motion) v12

## Key Directories
- `src/app`: App Router pages and layouts
- `src/components`: UI components
- `public`: Static assets
